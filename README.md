# 任务

- 使用 Bootstrap 响应式框架
- 至少 5 个不同外观效果的页面
- 包含下述组件中的8个： 导航栏、导航、轮播器、列表组、媒体对象、卡片、折叠展开面板、表格、表单、分页、按钮组、输入组、模态框、弹出框、信息提示框、loading动画。
- 提交资料： 源代码包（不超过5M）+说明文档

个人响应式博客 - 记录点滴生活
